# SimonSay-s-Game
Simon Says is a classic memory game where players must repeat a sequence of colours in the same order that the game presents them. The game starts by displaying a colour sequence to the player, which they must memorise and then repeat back by clicking on the corresponding colored buttons. The sequence becomes longer and more challenging as the player progresses through the levels.


## 🎮 How to Play Simon Says

1. Starting the Game: Press any key to start the game.

2. Gameplay: Watch the game present a sequence of colours.

3. Repeat the Sequence: Click on the buttons in the same order as the sequence presented by the game.

4. Level Up: Successfully repeat the sequence to advance to the next level.

5. Game Over: If you make a mistake and click the wrong button or click them out of order, the game ends, and you must start over.
